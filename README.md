# Food Ordering Website

A simple front-end shopping website built with HTML, CSS and Vanilla JavaScript.

## Features
- Product listing
- Shopping cart system
- User orders section
- Responsive design
- Dynamic product rendering using JSON data
- Mobile menu and UI interactions

## Technologies
- HTML5
- CSS3
- Vanilla JavaScript

## Project Structure
```bash
project/
│── index.html
│
├── js/
│   ├── main.js
│   └── menu.json
│
├── css/
│   └── style.css
│
├── images/
│
└── videos/
```

## Run
Just open `index.html` in your browser.
